# Project context

## Server

* Name: Last Survivor Vanilla+
* Platform: PlayStation 5
* Hosting: Nitrado
* Map: Chernarus
* Purpose: Temporary development and testing server

## Current files

The `legacy/` directory contains configuration files recovered from a previous version of the server.

These files are believed to contain custom modifications, but this has not yet been verified against fresh vanilla Chernarus files.

## Validation plan

Once a new Nitrado server is available:

1. Download the fresh vanilla Chernarus configuration files.
2. Store them separately from the legacy files.
3. Compare vanilla and legacy versions.
4. Identify custom changes.
5. Validate XML and JSON syntax.
6. Test confirmed changes on the live server.
7. Document the results for GitHub and the portfolio.
